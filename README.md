# Typescript Boilerplate

```bash
# Clean
yarn clean # remove build folder
yarn clean:coverage # remove coverage folder
yarn clean:all # remove build, coverage and node_modules folders
# Build
yarn build
# Start the script
yarn start
# Run unit tests
yarn test:unit
yarn test:unit:watch
# Linter (eslint)
yarn lint
# Formater (prettier)
yarn lint:fix
# Typescript Linter (eslint)
yarn lint:typescript
```
